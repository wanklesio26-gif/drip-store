👟 Loja Drip
Status: 🚀 Em desenvolvimento

O Loja Drip é um e-commerce focado em moda urbana (streetwear), desenvolvido como parte do curso de Front-end. O projeto foca em uma interface responsiva, rápida e com navegação fluida entre os produtos.

🛠 Tecnologias
As principais ferramentas utilizadas na construção deste projeto:

React (Biblioteca principal)

Vite (Build tool rápida para o desenvolvimento)

Tailwind CSS (Estilização via classes utilitárias)

React Router (Gerenciamento de rotas e navegação)

ESLint (Padronização e qualidade de código)

📁 Estrutura de Pastas
Baseado na organização do projeto:

src/: Contém os componentes, páginas e lógica da aplicação.

public/: Ativos estáticos como ícones e imagens.

tailwind.config.js: Configurações personalizadas de design.

vite.config.js: Configurações do ambiente de build.

🚀 Como rodar o projeto
Instale as dependências:

Bash
npm install
Inicie o servidor de desenvolvimento:

Bash
npm run dev
Acesse no navegador:
Acesse o endereço que aparecer no terminal (geralmente http://localhost:5173).
